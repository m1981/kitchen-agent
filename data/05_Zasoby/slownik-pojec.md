# Słownik pojęć – meble kuchenne i materiały płytowe

---

## Pojęcia podstawowe

**Korpus** – szkielet szafki kuchennej (boki, dno, góra, ścianka tylna), niewidoczny po zamontowaniu frontu i listew maskujących.

**Front** – widoczna, eksponowana ściana szafki; element dekoracyjny i funkcjonalny.

**Płyta wiórowa (PW)** – rdzeń z wiórów drzewnych sprasowanych żywicą, podstawowy materiał korpusowy.

**Płyta MDF** (Medium Density Fibreboard) – płyta z włókien drzewnych, jednorodna, gęsta, idealna do frezowania i lakierowania.

**Płyta HDF** (High Density Fibreboard) – jak MDF, ale większa gęstość; stosowana na ścianki tylne i plecy szafek.

**OSB** (Oriented Strand Board) – płyta z orientowanych wiórów; w meblach rzadko, częściej budownictwo.

**Okleina naturalna** – cienki fornir z drewna litego (0,5–1,5 mm) naklejany na rdzeń.

**Okleina syntetyczna (folia CPL/HPL)** – laminat papierowy nasycony żywicami, naklejany na rdzeń.

**CPL** (Continuous Pressure Laminate) – laminat niskonapięciowy, cienki (ok. 0,2–0,5 mm), stosowany przemysłowo.

**HPL** (High Pressure Laminate) – laminat wysokociśnieniowy (0,5–1,5 mm i grubszy), twardszy, bardziej odporny.

**Melamin / papier melamina** – papier dekoracyjny nasycony żywicą melaminowo-formaldehydową; po sprasowaniu tworzy powierzchnię płyty.

**Postforming** – technologia gięcia HPL na gorąco wokół zaokrąglonych krawędzi (np. blaty kuchenne).

**ABS** (Akrylonitryl-Butadien-Styren) – tworzywo do obrzeży; odporne na uderzenia, brak spoin przy klejeniu laserowym.

**PVC** – obrzeże z polichlorku winylu; tańsze od ABS, mniej ekologiczne.

**Obrzeże** – pasek materiału (ABS, PVC, fornir, HPL) naklejany na widoczne krawędzie płyty.

**Klejenie laserowe (Zero-Joint)** – technologia klejenia obrzeży aktywowanym laserem polimerem; brak widocznej linii kleju.

**Klejenie EVA / PUR** – kleje termotopliwe do obrzeży; PUR bardziej odporny na wilgoć i temperaturę.

**Format płyty** – standardowy wymiar arkusza, np. 2800×2070 mm, 2620×2070 mm.

**Grubość standardowa** – dla korpusów 16–18 mm; dla frontów 18–22 mm MDF.

**Klasa emisji E1 / E0,5 / CARB2** – normy emisji formaldehydu; E1 = standard europejski (≤ 0,1 ppm).

**Odporność na wilgoć (V313 / P3)** – parametr płyty wiórowej lub MDF dopuszczonej do środowisk wilgotnych (kuchnia, łazienka).

# Budowa płyt na korpus

## 1. Płyta wiórowa melaminowana (standard rynkowy)

```
┌─────────────────────────────────┐
│   Papier melamina (góra)        │  ← 60–120 g/m²
├─────────────────────────────────┤
│   Warstwa powierzchniowa        │  ← drobne wióry, wysoka gęstość
│   (fine face)                   │
├─────────────────────────────────┤
│   Rdzeń z wiórów grubych        │  ← 70–80% masy płyty
│   (core)                        │
├─────────────────────────────────┤
│   Warstwa powierzchniowa        │  ← symetria zapobiega wypaczeniu
│   (fine face)                   │
├─────────────────────────────────┤
│   Papier melamina (dół)         │  ← balans – ta sama gramatura
└─────────────────────────────────┘
```

**Typowe grubości:** 16 mm, 18 mm (korpus), 12 mm (ścianki boczne małych szafek), 8 mm (plecy).

**Kronospan:** linia **Kronobuild**, **Kronodesign** – płyty melaminowane z dekorem, klasa E1 lub E0,5. Produkty np. **D 2745**, **K 016**, kolekcje Stone, Wood, Uni.

**Zalety:** niska cena, duży wybór dekorów, dobra obrabialność.

**Wady:** wrażliwość krawędzi na wilgoć, możliwe delaminowanie przy złym klejeniu obrzeży.

---

## 2. Płyta wiórowa wilgocioodporna (zielony rdzeń, V313)

```
┌─────────────────────────────────┐
│   Papier melamina               │
├─────────────────────────────────┤
│   Rdzeń z wiórów + parafina     │  ← zielony barwnik = znacznik
│   + żywica MF/UF                │    wilgocioodporności
├─────────────────────────────────┤
│   Papier melamina               │
└─────────────────────────────────┘
```

**Zastosowanie:** szafki podumywalkowe, korpusy dolne przy zlewie kuchennym, meble łazienkowe.

**Kronospan:** oznaczenie **P3 / V313** – norma EN 312. Rdzeń barwiony na zielono jako identyfikacja.

---

## 3. Płyta MDF standard (do korpusów frezowanych, ramiaków)

```
┌─────────────────────────────────┐
│   Warstwa szlifierska           │  ← powierzchnia gładka, Ra < 1 µm
├─────────────────────────────────┤
│   Włókna drzewne                │
│   + żywica MUF                  │  ← jednorodna struktura na przekroju
│   (jednorodny przekrój)         │
├─────────────────────────────────┤
│   Warstwa szlifierska           │
└─────────────────────────────────┘
```

**Grubości:** 12, 16, 18, 22 mm.

**Kronospan:** linia **Kronolaf MDF**, **MDF Standard** – do frezowania profili, podkładu pod lakier.

# Budowa płyt na fronty

## 1. MDF lakierowany (front gładki lub frezowany)

```
┌─────────────────────────────────┐
│   Lakier poliuretanowy / UV     │  ← 2–4 warstwy, szlif między warstwami
│   (nawet 60–80 µm łącznie)     │
├─────────────────────────────────┤
│   Grunt / podkład               │
├─────────────────────────────────┤
│   MDF 18–22 mm                  │  ← frezowanie profili CNC
├─────────────────────────────────┤
│   Grunt + lakier (tył)          │  ← konieczny balans!
└─────────────────────────────────┘
```

**Kluczowe:** obustronne lakierowanie zapobiega wypaczeniu. Fronty bez balansu odkształcają się po kilku miesiącach.

---

## 2. Front MDF foliowany (folia PVC / CPL)

```
┌─────────────────────────────────┐
│   Folia PVC / CPL 0,2–0,4 mm   │  ← nadruk dekoru + warstwa ochronna
├─────────────────────────────────┤
│   Klej reaktywny (membrana)     │  ← technologia membranowa (vacuum)
├─────────────────────────────────┤
│   MDF 18 mm z frezem            │  ← możliwy profil 3D
└─────────────────────────────────┘
```

**Technologia membranowa (vacuum pressing):** folia owijana pod próżnią – wchodzi w każdy profil frezu CNC.

---

## 3. Front HPL (laminat wysokociśnieniowy)

```
┌─────────────────────────────────┐
│   HPL 0,5–1,5 mm                │  ← wierzchnia warstwa dekoracyjna
├─────────────────────────────────┤
│   Klej PVAc / PUR               │
├─────────────────────────────────┤
│   MDF lub PW 18–19 mm           │  ← rdzeń
├─────────────────────────────────┤
│   Klej + HPL (balans)           │  ← HPL balansu – często plain/kraft
└─────────────────────────────────┘
```

**Kronospan:** linia **Krono HPL**, np. **K354 PE**, kolekcje SuperMatt, Gloss, Stone, Metal.

**Odporność:** wysoka na zarysowania, wilgoć, temperaturę – idealne do kuchni.

---

## 4. Front akrylowy (wysoki połysk)

```
┌─────────────────────────────────┐
│   Panel akrylowy 1–3 mm         │  ← PMMA, polerowany
├─────────────────────────────────┤
│   Klej dwuskładnikowy / taśma   │
├─────────────────────────────────┤
│   MDF 16–18 mm                  │
├─────────────────────────────────┤
│   Balans (folia / HPL)          │
└─────────────────────────────────┘
```

**Uwaga:** wrażliwość na zarysowania przy eksploatacji, trudność naprawy.

---

## 5. Front z okleiną naturalną (fornir)

```
┌─────────────────────────────────┐
│   Lakier / olej                 │
├─────────────────────────────────┤
│   Fornir 0,5–1,5 mm             │  ← dąb, orzech, jesion, mahoń...
├─────────────────────────────────┤
│   Podkład (papier/folia)        │
├─────────────────────────────────┤
│   MDF lub PW 18 mm              │
├─────────────────────────────────┤
│   Fornir balansu (tył)          │
└─────────────────────────────────┘
```
